Info on SAS University

SAS University Edition is a FREE version of SAS that you can download, install and run for FREE.

SAS University

SAS University is a free version of SAS which run through a "virtual machine" (Oracle VM Virtual Box). This "virtual machine" creates the environment within which the SAS University application runs. This application of SAS is MUCH MORE USER FRIENDLY with GUI's (graphical user interface) with pull down menus. The interface is analogous to the menu driven interface of SPSS. However, within these SAS University menus you can easily select variables and specify SAS options on the fly without having to dig through the SAS help system! Similar to SPSS, once the menus are configured with your variable selections and options for the procedure you've selected, the SAS code is written and COMMENTED for you ON THE FLY. This is a conderful was to learn SAS.

That said, admittedly setting up SAS University does require some time and effort to set-up and once set-up and running it does run slower than fully licensed SAS that you pay for. But SAS University is FREE and full featured. Not all SAS functions are easily available through the menu system, but nearly all SAS functions can still be called from the SAS program window as usual.
